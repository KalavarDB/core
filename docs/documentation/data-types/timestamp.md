# Timestamp
Stores a timestamp for future reference

### Information
- Format: `TZ-YYYYMM DD-HH:MM:SS.xxx`
- Encoding: Binary

### Breakdown
|Example|Timezone|Year|Month|Day|Hour|Min|Sec|extended|
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
|UTC+0-2021 01 01-00:00:00.000 |UTC+0|2021|Jan|01|0|0|0|000|
